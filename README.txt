★elona『妹』ゴースト

本作品は、noa氏製作のフリーゲーム『elona』の二次創作ゴーストです。
elonaに関するネタと、elonaヴァリアントである「omake」のネタが会話等に入っています。

製作元クレジットをreadmeもしくはそれに相当するファイルに表記してあれば、改変は自由、改変しての再配布もokです。
改変無しの再配布につきましては、再配布を行うページにこちら(https://github.com/tolia512/elona_imouto_ghost)


#####################################################################

機能（予定）
　・ランダムなお喋り—お喋りする内容の設定可能。elonaの会話だけをしてほしい方から、実生活への応援や癒しがほしい方まで用途に合わせて。
　・時報
　・アラーム
　・着せ替え
　・撫でたりつついたりすると反応

設定（予定）
　・妹に名前をつける（初期ネームは「妹」）
　・ゴーストのサイズ変更

複雑な機能（案）
　・会話モード
　・スケジューラ
　・アイテムを渡す（お遊び機能）
　・願い（性転換、神（サブキャラ）の呼び出しなど、elona的なおまけ機能）
